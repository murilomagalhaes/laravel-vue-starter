## About:
A boilerplate Laravel 8 app set up with Bootstrap 5, VueJS 2 and a custom template with responsive/collapsable side and top menus. 

## Bundled With:

- Laravel 8
- Laravel Debugbar
- Bootstrap 5.1
- Vue 2
- Custom template w/ responsive side and top menu bars

- Clone the repo
- Run `comopser install`
