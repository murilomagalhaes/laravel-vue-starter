## About:
A boilerplate Laravel 8 app set up with Bootstrap, VueJS and a custom template with responsive side and top menu bars. 

## Bundled With:

- Laravel 8
- Laravel Debugbar
- Bootstrap 5.1
- Vue 2
- Custom template w/ responsive side and top menu bars
